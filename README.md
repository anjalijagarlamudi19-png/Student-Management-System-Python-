## Student-Management-System-Python-

## 📌 Project Overview

The Student Management System is a Python console-based application used to manage student records. It allows users to add, update, search, delete, and display student details along with marks, grades, and class statistics.

This project also includes a simple login system for authentication.

## 🚀 Features

-  Login Authentication System
-  Add Student Details
-  Display All Students
-  Search Student by Roll Number
-  Update Student Information
-  Delete Student Record
-  Find Highest Marks Student
-  Calculate Class Average
-  Sort Students by Total Marks
-  Automatic Grade Calculation
  
## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- Lists & Dictionaries
- Functions & Loops
